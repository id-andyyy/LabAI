Пользователь сообщает тему дополнительного исследования для отчёта.

## Что делать

1. Прочитай report.md и CLAUDE.md.
2. Изучи тему исследования (указана ниже).
3. Если не хватает знаний — выполни поиск в авторитетных интернет-источниках. Не выдумывай.
4. Добавь в report.md раздел дополнительного исследования:
   - Это должен быть анализ, а не просто изложение фактов.
   - Содержательные умозаключения, связь с основной темой лабораторной работы.
   - Оформление по ГОСТ (правила из CLAUDE.md).
   - Чистый текст, без markdown-форматирования.
5. Сообщи в чате, что раздел добавлен, и кратко опиши его содержание.
6. Напомни, что для доработки можно использовать /improve.
