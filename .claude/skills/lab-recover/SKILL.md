Пользователь хочет вернуть компьютер в состояние до выполнения лабораторной работы.

## Что делать

1. Прочитай instructions.md (только его, report.md не трогай).
2. Определи, что в ходе работы было изменено в системе: созданы файлы/папки, включены службы, изменена конфигурация сети, реестра, брандмауэра, добавлены пользователи и т.д.
3. Исключи из списка всё, что уже было указано в инструкции как шаг по откату (пользователь это уже сделал).
4. Исключи из списка установку/запуск приложений, которые использовались для выполнения работы — это не изменение конфигурации системы.
5. Составь список оставшихся действий по откату.

## Как ответить

Ответ — только в чате, кратко и по делу, без лишних объяснений. Формат:

- Если откатывать нечего: одна фраза — «Ничего делать не нужно, конфигурация системы не менялась.»
- Если есть что откатить: нумерованный список конкретных действий. Без вступлений, без пояснений — только действия.

Не трогай и не изменяй никакие файлы проекта.
